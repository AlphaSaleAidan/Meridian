System Architecture
Three-tier storage, 50 AI agents, DeepSeek V3 reasoning (671B MoE), edge vision processing

Data Flow

1. POS webhooks (Square/Clover) → API ingest
2. Edge cameras → structured analytics (no raw video)
3. 50 AI agents analyze per-org (Karpathy 5-phase)
4. Cross-reference engine fuses POS + Vision + Journeys
5. Swarm trainer learns patterns across all merchants
6. Cold storage compresses → 6 tiers → R2 offload
7. Resale API packages anonymized data for buyers
