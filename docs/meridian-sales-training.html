Camera Setup: Auto-Discover

EASIEST

Works with WiFi/wired cameras supporting ONVIF/UPnP. System scans the network and finds cameras automatically.

1

Verify Network Access

Meridian device must be on the same WiFi network as cameras. Go to Cameras → Connect Camera → Auto-Discover.

2

Run Network Scan

Click "Scan Network." Wait 15–30 seconds. Cameras appear with IP, manufacturer, model.

3

Authenticate & Configure

Select cameras, enter admin credentials (default: admin/admin or admin/12345). Name each camera, set zones, enable privacy masking.

⚠️ NO CAMERAS FOUND?
Check cameras are powered on and on the same network. Some need ONVIF enabled in their settings. If discovery fails → fall back to RTSP Manual.
💡 RTSP MANUAL FALLBACK
Enter camera's stream URL directly. Find IP in router settings. Format by brand:
• Hikvision: rtsp://user:pass@ip:554/Streaming/Channels/101
• Dahua: rtsp://user:pass@ip:554/cam/realmonitor?channel=1
• Reolink: rtsp://user:pass@ip:554/h264Preview_01_main
• Generic: rtsp://user:pass@ip:554/stream1
